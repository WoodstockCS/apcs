Are you ready?

Before starting this course you will need:

  • some basic programming skills: you should start the year with a good grasp of some basic fundamentals in programming, like functions, booleans, and lists
  • a pro-active approach to self-learning: when you feel stuck, you should feel comfortable finding help in books or online resources
  • good team skills: contrary to popular belief, programming is not a solitary task. you should be able to work together with -- and serve as a resource for -- your fellow programmers in the course
  • good work habits: this course is not easy, so you should be willing to work hard and effectively

Take another look at that first one, about programming skills. This is not an intro class, so you need to have some experience before the class starts. Most students get this experience by taking Computer Programming at WUHS, but some students get this experience elsewhere. The following steps show you how to check whether your experience has given you the pre-requisite skills.
